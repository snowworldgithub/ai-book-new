# Speckit + Docusaurus Documentation

Welcome to the documentation system for the AI-Native Physical AI & Humanoid Robotics textbook. This system combines Speckit for automated documentation generation with Docusaurus for a powerful documentation platform.

## What is Speckit + Docusaurus?

Speckit + Docusaurus is a documentation system that automatically generates specifications and documentation from your codebase. It leverages:

- **Speckit**: Automated documentation generator that extracts information from code comments, config files, and API definitions
- **Docusaurus**: Modern static site generator optimized for documentation

## Key Features

- **Automated Documentation Generation**: Generate specs from code comments and definitions
- **Multi-Format Support**: Documentation, guides, API references, and specifications
- **Component Playground**: Interactive component examples using React Live
- **AI Integration**: Ready for integration with RAG chatbot and personalization systems
- **Internationalization**: Built-in support for multiple languages including Urdu
- **Versioning**: Documentation versioning system
