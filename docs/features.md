# Features

The Speckit + Docusaurus system provides a comprehensive set of features for creating and maintaining documentation for the AI-Native Physical AI & Humanoid Robotics textbook.

## Automated Documentation Generation

Speckit automatically extracts documentation from:
- Code comments (JSDoc, TS Doc)
- Configuration files
- API definitions
- Markdown files

This ensures your documentation stays up-to-date with your codebase.

## Multi-Format Documentation

The system supports multiple types of documentation:
- **Guides**: Step-by-step tutorials
- **API References**: Auto-generated API documentation
- **Specifications**: Generated from code
- **Conceptual Docs**: High-level explanations

## Component Playground

Interactive component examples using React Live allow users to:
- See components in action
- Modify code and see results
- Understand component usage patterns
- Test different configurations

## AI Integration Ready

Built-in hooks for AI features:
- RAG chatbot integration
- Content personalization
- Urdu translation system
- Claude subagents and skills

## Internationalization

Support for multiple languages:
- English (primary)
- Urdu (for translation system)
- Extensible to other languages
- Right-to-left text support

## Versioning System

Manage documentation versions:
- Version-specific documentation
- Automatic redirects for moved content
- Clear version indicators
- Easy rollback capabilities
