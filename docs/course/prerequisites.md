# Prerequisites

To successfully complete this course, students should have the following background knowledge and technical requirements.

## Academic Prerequisites

- **Mathematics**:
  - Linear algebra (vectors, matrices, transformations)
  - Calculus (derivatives, integrals, optimization)
  - Probability and statistics
  - Basic differential equations

- **Computer Science**:
  - Programming experience in Python and C++
  - Understanding of data structures and algorithms
  - Basic knowledge of machine learning concepts
  - Familiarity with version control systems (Git)

- **Engineering**:
  - Basic physics (mechanics, dynamics)
  - Introduction to control systems
  - Basic electronics concepts

## Technical Requirements

### Software
- Modern Linux distribution (Ubuntu 20.04 LTS or later recommended)
- Windows Subsystem for Linux (WSL2) if using Windows
- macOS with development tools

### Development Tools
- ROS 2 Humble Hawksbill or later
- Python 3.8 or later
- C++ compiler (GCC 9 or later)
- Git version control
- Docker (for simulation environments)
- Basic Linux command line proficiency

### Hardware Access
While personal robotics hardware is not required, access to:
- Robotic simulation environments (Gazebo, Isaac Sim)
- Shared robotics lab equipment (for advanced projects)
- Standard computer hardware capable of running simulations

## Recommended Preparation

Students without the above prerequisites are encouraged to complete the following preparation modules:

- Mathematics for Robotics (self-paced) - covered in the intro section
- Python for Robotics (2-week intensive) - refer to getting started guide
- Introduction to Control Systems (self-paced) - available in best practices

## Assessment

Students will complete a prerequisites assessment during the first week to identify any gaps in required knowledge and receive personalized recommendations for additional preparation.
