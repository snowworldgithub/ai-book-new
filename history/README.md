# History

This directory contains historical records of the development process:

- `prompts/` - Prompt History Records (PHRs) for all development activities
  - `constitution/` - PHRs related to constitution development and changes
  - `general/` - General PHRs for other development activities
  - `<feature-name>/` - Feature-specific PHRs (when features are developed)

These records maintain traceability and context for all changes made to the course content and infrastructure.
